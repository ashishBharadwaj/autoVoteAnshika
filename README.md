# autoVoteAnshika
Automatically vote in MyCuteBaby photo contest 


# How to Use :

  ** Prequisite: Git, Node.js
  
- Clone the repository:

```bash
  $ git clone https://github.com/ashishBharadwaj/autoVoteAnshika.git
```

- Install the dependencies:

```bash
$ npm install
```

- Start script

```bash
$ npm start
```
